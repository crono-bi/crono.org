
# Crono$DatabaseUsers

Devuelve el listado de usuarios de todas las bases de datos del servidor.

```
SELECT *
FROM Crono$DatabaseUsers
```

La consulta incluye estas columnas:

- DatabaseName
- Username
- TypeDescription
- DefaultSchemaName
- CreationDate
- ModifiedDate
- AuthenticationType
- [Login]
- LoginType
- DefaultDatabaseName
- DefaultLanguage
- LoginDisabled
