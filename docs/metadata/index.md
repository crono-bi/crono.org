# Crono Metadata

Crono Metadata es la aplicación para construir la capa semántica de Crono

## Características

### Business Intelligence fácil
La característica principal de Crono es la facilidad de uso. Así de simple. ❤️

### Visión de negocio
El catálogo muestra los datos con la visión de negocio que los analistas necesitan.🧑‍🦰

### Potente
Con toda la potencia y funcionalidades de las mejores herramientas de BI. 🛠️
