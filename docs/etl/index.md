# Crono ETL

Una herramienta ETL especializada en proyectos DWH/ETL

## Características

### Crono SQL
Crono ETL utiliza el lenguaje Crono SQL para crear los procesos de carga de un DWH.🛠️

### SQL con superpoderes
Un lenguaje como el SQL, pero más fácil de escribir, más mantenible y más escalable. ❤️

### Generador de SQL
No es una BBDD ni una caja negra que ejecuta procesos ETLs. Crono SQL es nada más (y nada menos) un generador de código SQL. 🥇
